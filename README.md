# im-netology-hw

Здесь собираю практические работы по системному администрированию, Linux, инфраструктуре и DevOps из курса Нетологии.

В каждой папке — отдельная работа: краткое описание задачи, решение, конфигурации, скрипты и документация.

## Работы

| Работа | Тема | Что внутри |
|---|---|---|
| [netology-032-yandex-cloud](netology-032-yandex-cloud/README.md) | Terraform, Yandex Cloud, Ansible | инфраструктура, inventory, настройка nginx, проверка результата |
| [netology-033-gitlab](netology-033-gitlab/README.md) | GitLab CI, GitLab Runner, Docker | runner, pipeline, Go tests, сборка бинарника, Docker build, скриншоты |

## Как читать

Откройте `README.md` конкретной работы по ссылке из таблицы: там собраны основные шаги, команды проверки и ссылки на дополнительные файлы.
